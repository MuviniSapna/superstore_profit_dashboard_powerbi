## Project Overview
Developed an interactive Power BI dashboard using the Superstore dataset to analyze sales, profit, quantity, and regional performance.

## Features
- KPI Cards for Sales, Profit, and Quantity
- Profit Analysis by Region, Segment, and Category
- Yearly and Quarterly Trend Analysis
- Interactive Filters and Slicers
- Drill-down Functionality for Better Insights

## Tools & Technologies
- Power BI
- DAX
- Power Query
- Excel

## Key Insights
- Identified high-performing regions and categories
- Compared profitability across business segments
- Analyzed yearly and quarterly growth trends
